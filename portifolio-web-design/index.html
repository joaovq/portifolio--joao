Sobre mim

HTML5

O Hyper-Text Markup Language 5, ou HTML5, é uma linguagem de marcação que ajusta os elementos de uma página da web. É a partir dele que organizamos os elementos de texto, imagens, icones, links que podemos e desejamos ter em uma página.

CSS3

O Cascade Style Sheet 3, ou CSS3 como é chamado popularmente, é uma ferramenta utilizado para "estilizar" os websites e web-apps feitos atualmente. É com ele que você pode colocar animações, cores que melhoram a experiência do usuário em uma ou mais páginas.

Java

A linguagem Java é uma das mais utilizadas no mercado e uma das mais seguras e robustas para você utilizar nos seus sistemas. Através do framework Spring Boot, é possível criar formulas seguras e de fácil manutenção.
